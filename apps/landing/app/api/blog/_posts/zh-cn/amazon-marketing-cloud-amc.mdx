# 什么是亚马逊AMC？

![Amazon Marketing Cloud AMC](/images/blog/amazon-marketing-cloud-amc.jpeg)

[亚马逊营销云（AMC）](https://www.revns.com/blog/post/amazon-amc-vs-dsp-audience)是一款专为亚马逊广告设计的高级[分析工具](https://www.revns.com/blog/post/master-amazon-seller-analytics)。它使广告主能够探索大规模数据集，获取有关受众行为的宝贵洞察，并优化多渠道广告活动效果。您可以将其视为数字广告指挥中心，为您的营销工作提供全面的可视性。

## 亚马逊营销云文档

要有效利用AMC，需要了解其核心组件：

### 数据集成和分析
亚马逊营销云（AMC）是一个安全的、基于云的"清洁室"环境，允许广告主分析广告活动效果、了解受众行为并优化广告策略。它通过提供一个安全空间实现这一目标，多个参与方可以在该空间中共同分享和分析数据，同时保持严格的隐私和安全标准。在此环境中，来自不同来源的数据可以以匿名和汇总的方式进行组合，确保不会暴露或共享任何个人身份信息（PII）。

AMC通过以下方式帮助广告主：
*   **上传和集成第一方数据** AMC允许广告主将自己的第一方数据与亚马逊的广告数据集成。这些数据上传到AMC环境中，以匿名和汇总的方式进行处理。AMC确保不会暴露任何个人用户数据，维持严格的数据隐私标准。
*   **访问跨渠道分析** AMC使品牌能够分析跨渠道广告活动效果，了解受众行为并优化广告。它提供多个数据集（包括广告主和亚马逊广告数据）的事件级数据的可定制报告。这提供了更全面的广告活动效果视图。
*   **维护客户隐私合规** AMC防止购物者被单独识别，不允许访问被分析购物者的任何个人身份信息（PII）。每个购物者都通过一系列设计用于保护购物者隐私的匿名"user_id"代码进行识别。
*   **从复杂数据集生成可操作洞察** AMC提供了为广告主及其合作伙伴创建清洁室环境所需的基础设施。在AMC的清洁室环境中，广告主可以运行基于SQL的自定义查询来分析数据集、发现洞察并优化其广告活动。结果始终是汇总的，不能显示任何特定用户信息，确保数据保持安全和隐私合规。

### 广告活动优化的关键功能

1.  **高级受众细分**
    *   基于浏览模式、购买频率和广告互动创建自定义受众细分
    *   分析跨平台的购买行为和广告互动以完善定位策略
    *   开发定向消息策略以提升广告相关性并增加转化率

2.  **跨渠道分析**
    *   监控亚马逊平台（如亚马逊中国、Fire TV和Alexa）上的客户互动
    *   了解客户旅程以识别有效接触点
    *   基于跨渠道数据优化广告投放和时机以最大化广告活动ROI

3.  **基于SQL的自定义报告**
    *   使用SQL查询自定义报告以关注与业务目标相关的特定指标
    *   跟踪关键性能指标（KPI），如转化率、广告覆盖率和归属于特定广告组的销售额
    *   在ASIN级别分析效果以改善获得最多关注的产品列表
    *   计算广告活动投资回报率（ROI）并基于效果数据优化预算分配

## 亚马逊营销云的实际应用

### 购物车放弃解决方案

亚马逊营销云（AMC）通过提供工具来分析客户为什么在完成购买前离开，从而使品牌能够解决购物车放弃问题。通过AMC，品牌可以获得关于放弃模式的洞察，创建定向重定向广告活动，并开发旨在鼓励客户返回并完成购买的个性化消息。AMC还允许测试各种促销策略，以确定提高转化率的最有效激励措施。

### 广告频率优化

AMC提供关于最佳广告展示频率的关键洞察，帮助品牌了解客户在转化前需要看到广告的次数。通过分析不同渠道的客户转化模式，AMC提供全面的广告效果视图。它支持渠道特定的效果跟踪，识别推动转化最有效的渠道，并提供广告疲劳指标以防止过度曝光和潜在的客户疏远。

### ASIN效果跟踪

AMC通过跟踪个别ASIN（亚马逊标准识别号）的效果，实现详细的产品级分析。这有助于识别高转化产品，使品牌能够将广告努力集中在最有利可图的商品上。通过AMC，品牌可以通过将资源引导到具有最高转化潜力的产品来优化广告支出分配，并通过识别可以改进产品列表以吸引更多客户的领域来提高列表效果。

## 使用亚马逊AMC的优势

1.  **数据驱动决策**
    *   AMC提供对**全面分析**的访问，为广告主提供关于其广告活动和受众行为的更好洞察。通过分析趋势和模式，用户可以识别改进机会。
    *   虽然AMC专注于回顾性分析而不是**实时效果跟踪**，但它允许用户衡量广告活动的增量影响并区分有机和广告驱动的销售。用户可以依靠其他[亚马逊工具](https://www.revns.com/blog/post/amazon-fba--revenue-calculator-free)进行即时优化。
    *   AMC促进**可操作洞察生成**，帮助制作与受众产生共鸣的广告活动。AI协助工具简化了传统上复杂的AMC SQL编写过程，充当数据分析合作伙伴。
    *   通过数据洞察，用户可以做出基于事实的战略决策，导致**基于证据的策略制定**。

2.  **提升广告活动效果**
    *   来自AMC的行为数据完善受众细分并**提高定位准确性**。用户可以根据其行为和偏好为特定客户细分定制内容和广告。
    *   AMC帮助广告主识别表现不佳的广告活动并重新分配预算，导致**更好的预算分配**。通过评估哪些渠道提供最佳效果指标，用户可以决定如何分配其营销预算。
    *   通过优化广告排序和衡量对转化的影响，AMC有助于实现**更高的转化率**。
    *   AMC洞察使广告主能够将预算重新分配到表现良好的广告活动，**提高广告支出回报率**（ROAS）。AMC实现的跨渠道策略可以带来25%的ROAS改善。

3.  **隐私合规分析**
    *   AMC通过匿名化和汇总数据确保**安全数据处理**，遵守全球法规并为分析创建安全环境。敏感信息永远不会暴露。
    *   AMC使用**匿名数据处理**，显示至少100个用户组的汇总数据而不是提供用户级数据。这种汇总使品牌能够在保护其隐私的同时获得客户行为洞察。
    *   AMC遵守GDPR、CCPA和其他全球隐私法规，确保**监管合规**。
    *   AMC保护客户信息，因为它只接受假名化信息，用户信号不能被亚马逊导出或访问。用户只能访问来自AMC的汇总、匿名输出。

## 开始使用亚马逊营销云

### 先决条件
1.   **显著的广告支出** 虽然没有普遍强制的最低预算，但通常需要大约5,000美元或更多的实质性年度广告预算。
2.   **技术SQL能力** SQL（结构化查询语言）熟练度对于在AMC内查询和分析数据至关重要，尽管亚马逊为常见用例提供预建查询。
3.   **强大的第一方数据** 集成高质量的第一方数据丰富了从AMC获得的洞察，实现更有针对性和有效的分析。
4.   **对数据驱动营销的承诺** 致力于使用数据来指导营销策略和优化广告活动对于最大化AMC的好处至关重要。

### 实施步骤
1.  **在亚马逊验证资格** 确保您的组织满足必要标准，如拥有亚马逊DSP服务协议或在过去28天内使用DSP运行广告活动。但是，最新更新表明所有亚马逊广告主现在都有资格使用AMC。
2.  **联系亚马逊代表** 联系亚马逊代表以启动注册流程并讨论具体要求。
3.  **设置AMC仪表板** 配置AMC仪表板以可视化和分析数据，可能需要Amazon Web Services（AWS）账户。
4.  **配置数据集成** 通过[API](https://www.revns.com/blog/post/amazon-advertising-api-documentation)、事件管理器或集成信号管理工具集成相关数据源，包括亚马逊广告活动、第三方提供商和您自己的第一方数据。
5.  **开始运行SQL查询** 利用SQL从集成数据中提取可操作洞察，优化广告活动策略并为业务决策提供信息。
6.  **监控和优化广告活动** 持续监控广告活动效果并利用来自AMC的洞察完善定位、预算分配和创意策略。
