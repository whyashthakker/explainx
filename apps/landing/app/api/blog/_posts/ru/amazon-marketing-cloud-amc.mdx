# Что такое Amazon AMC?

![Amazon Marketing Cloud AMC](/images/blog/amazon-marketing-cloud-amc.jpeg)

[Amazon Marketing Cloud (AMC)](https://www.revns.com/blog/post/amazon-amc-vs-dsp-audience) - это сложный [инструмент анализа](https://www.revns.com/blog/post/master-amazon-seller-analytics) данных, специально разработанный для Amazon Advertising. Он позволяет рекламодателям исследовать большие наборы данных, получать ценные инсайты о поведении аудитории и оптимизировать эффективность кампаний по нескольким каналам. Представьте его как ваш командный центр цифровой рекламы, обеспечивающий комплексную видимость ваших маркетинговых усилий.

## Документация Amazon Marketing Cloud

Чтобы эффективно использовать AMC, необходимо ознакомиться с его основными компонентами:

### Интеграция и Анализ Данных
Amazon Marketing Cloud (AMC) - это безопасная облачная среда "чистой комнаты", которая позволяет рекламодателям анализировать эффективность кампаний, понимать поведение аудитории и оптимизировать рекламные стратегии. Это достигается путем предоставления безопасного пространства, где несколько сторон могут совместно делиться и анализировать данные, сохраняя при этом строгие стандарты конфиденциальности и безопасности. В этой среде данные из разных источников могут быть объединены анонимным и агрегированным способом, гарантируя, что никакая личная идентифицируемая информация (PII) не раскрывается и не передается.

AMC помогает рекламодателям несколькими способами:
*   **Загрузка и интеграция собственных данных** AMC позволяет рекламодателям интегрировать свои собственные данные с рекламными данными Amazon. Эти данные загружаются в среду AMC, где они обрабатываются анонимным и агрегированным способом. AMC гарантирует, что никакие индивидуальные пользовательские данные не раскрываются, поддерживая строгие стандарты конфиденциальности данных.
*   **Доступ к кросс-канальной аналитике** AMC позволяет брендам анализировать эффективность кросс-канальных кампаний, понимать поведение аудитории и оптимизировать рекламу. Он предлагает настраиваемую отчетность по данным на уровне событий по нескольким наборам данных, включая как данные рекламодателя, так и данные Amazon Advertising. Это обеспечивает более целостное представление об эффективности кампании.
*   **Поддержание соответствия конфиденциальности клиентов** AMC предотвращает индивидуальную идентификацию покупателей и не позволяет доступ к любой личной идентифицируемой информации (PII) о анализируемых покупателях. Каждый покупатель идентифицируется через серию анонимизированных кодов "user_id", которые разработаны для защиты конфиденциальности покупателя.
*   **Генерация действенных инсайтов из сложных наборов данных** AMC предоставляет инфраструктуру, необходимую для создания среды чистой комнаты для рекламодателей и их партнеров. В среде чистой комнаты AMC рекламодатели могут выполнять пользовательские SQL-запросы для анализа наборов данных, обнаружения инсайтов и оптимизации своих кампаний. Результаты всегда агрегированы и не могут раскрывать никакую конкретную пользовательскую информацию, обеспечивая безопасность данных и соответствие конфиденциальности.

    *   Оптимизация размещения и времени рекламы на основе кросс-канальных данных для максимизации ROI кампании.
3.  **Пользовательская Отчетность на Основе SQL**

    *   Отслеживание Ключевых Показателей Эффективности (KPI), таких как коэффициенты конверсии, охват рекламы и продажи, приписываемые конкретным рекламным группам.

## Реальные Применения Amazon Marketing Cloud

### Решение Проблемы Брошенных Корзин

Amazon Marketing Cloud (AMC) позволяет брендам решать проблему брошенных корзин, предоставляя инструменты для анализа причин, по которым клиенты уходят без завершения покупки. Через AMC бренды могут получать инсайты о паттернах отказа, создавать целевые кампании ретаргетинга и разрабатывать персонализированные сообщения, предназначенные для поощрения клиентов вернуться и завершить свои покупки. AMC также позволяет тестировать различные промо-стратегии для определения наиболее эффективных стимулов для повышения коэффициентов конверсии.

### Оптимизация Частоты Рекламы

AMC предоставляет критически важные инсайты об оптимальной частоте показа рекламы, помогая брендам понять, как часто клиенты должны видеть рекламу перед конверсией. Анализируя паттерны конверсии клиентов по различным каналам, AMC обеспечивает комплексное представление об эффективности рекламы. Он позволяет отслеживать эффективность по конкретным каналам, определяя наиболее эффективные каналы для привлечения конверсий, и предлагает индикаторы рекламной усталости для предотвращения чрезмерного воздействия и потенциального отчуждения клиентов.

### Отслеживание Эффективности ASIN

AMC позволяет проводить детальный анализ на уровне продуктов путем отслеживания эффективности отдельных ASIN (Amazon Standard Identification Numbers). Это помогает идентифицировать продукты с высокой конверсией, позволяя брендам сфокусировать свои рекламные усилия на наиболее прибыльных товарах. С помощью AMC бренды могут оптимизировать распределение рекламных расходов, направляя ресурсы на продукты с наивысшим потенциалом конверсии, и улучшать эффективность листингов, точно определяя области, где листинги продуктов могут быть улучшены для привлечения большего количества клиентов.
