# 利益戦争：FBA vs FBM – 勝利の戦略を見つけよう！

![FBA vs FBM 勝利の戦略を見つけよう](/images/blog/fba-vs-fbm-find-your-winning-strategy.jpeg)

Amazonビジネスの成功は、適切なフルフィルメント方式の選択に大きく依存します。ビジネスを始めたばかりでも、既存の運営を最適化しようとしている場合でも、フルフィルメント by マーチャント（FBM）とフルフィルメント by Amazon（FBA）の違いを理解することは重要です。この包括的なガイドは、あなたのビジネスに関する十分な情報に基づいた決定を下すのに役立ちます。

## はじめに

フルフィルメント by Amazon（FBA）では、出品者は商品をAmazonのフルフィルメントセンターに送り、Amazonが保管、梱包、配送、さらにカスタマーサービスまでを担当します。このモデルでは、出品者はAmazonの広範な物流ネットワークの恩恵を受け、Primeの対象となる迅速な配送オプションを提供できるため、商品の可視性と販売可能性を大幅に向上させることができます。ただし、FBAにはフルフィルメントや保管費用など、様々な手数料が伴い、利益率に影響を与える可能性があります。

対照的に、フルフィルメント by マーチャント（FBM）では、出品者が在庫と配送プロセスを完全にコントロールします。フルフィルメントのすべての側面を自社で管理するか、サードパーティの物流プロバイダーを通じて管理します。この方式では、独自の配送要件の取り扱いにおいてより大きな柔軟性があり、FBAに関連する多くの手数料を避けられるため、より良い利益率につながる可能性があります。ただし、FBMではより多くの実践的な管理が必要で、FBAと同レベルのカスタマーサービスや速度を提供できない場合があります。

## FBMとは？

フルフィルメント by マーチャント（FBM）は、出品者が商品の保管、梱包、顧客への直接配送の全責任を負うAmazonのフルフィルメント方式です。Amazonが物流を担当するフルフィルメント by Amazon（FBA）とは異なり、FBMでは出品者が注文処理プロセス全体を独自に、またはサードパーティの物流プロバイダーを通じて管理します。

### FBMの仕組み

- **在庫管理**: 出品者は自社の施設で在庫を維持します。在庫レベルの追跡、必要に応じた商品の補充、商品リストの正確性と最新性の確保に責任を負います。

- **注文処理**: 注文を受けると、出品者は商品を梱包し、顧客の住所への配送を手配します。これには、タイムリーな配達を確保するための信頼できる配送戦略が必要です。

- **カスタマーサービス**: FBM出品者は、顧客からの問い合わせ、返品の処理、満足のいく買い物体験の確保も担当します。これにより、効果的なカスタマーサービススキルとリソースの需要が高まる可能性があります。

### FBMの利点
1. **完全なコントロール**: 商品の保管、梱包、配送方法を自分で決定できる
2. **コスト効率**: 特に大型商品の場合、FBAと比べて手数料が低い
3. **柔軟な在庫管理**: 在庫レベルと季節商品をより良くコントロールできる
4. **直接的な顧客コミュニケーション**: 顧客のニーズに即座に対応可能

### FBMの欠点
1. **作業負荷が高い**: フルフィルメントのすべての側面を自己管理
2. **Prime対象の制限**: Primeの配送資格を得るのが難しい
3. **複雑なスケーリング**: 重要なインフラなしでの成長が困難
4. **配送時間の変動**: 配送速度は自社の能力に依存

## FBM収益性の計算式

FBMでの潜在的な利益を計算するには、以下の式を使用します：

```
{純利益} = {販売価格} - ({商品原価} + {配送費} + {Amazon手数料} + {運営費用})
```

### 内訳：
- **販売価格**: Amazonで商品を販売する価格。
- **商品原価（COGS）**: 商品の製造または購入にかかる総費用（製造費用と自社への配送費用を含む）。
- **Amazon手数料**: フルフィルメントサービスに対してAmazonが請求するすべての手数料（商品のサイズと重量に応じて変動する保管料と処理料を含む）。
- **Amazonへの配送**: 在庫をAmazonのフルフィルメントセンターに送る費用。
- **運営費用**: 広告、梱包、その他の間接費など、ビジネス運営に関連する追加費用。

## FBAとは？

フルフィルメント by Amazon（FBA）は、出品者がAmazonの広範な物流ネットワークを活用して、商品の保管、梱包、顧客への直接配送を行えるサービスです。FBAでは、出品者は在庫をAmazonのフルフィルメントセンターに送り、Amazonが保管、梱包、配送、カスタマーサービスを含むフルフィルメントプロセス全体を担当します。このモデルは出品者の運営を簡素化するだけでなく、Amazon Primeの顧客へのアクセスも提供し、商品の可視性と販売可能性を向上させます。FBAを利用することで、出品者はAmazonの迅速で信頼性の高い配送の評判を活かしながら、ビジネスの成長に集中できます。さらに、FBAは出品者が在庫を効率的に管理し、運営を合理化するのに役立つツールとリソースを提供し、eコマースプレゼンスを最適化しようとする新規および既存の出品者にとって魅力的なオプションとなっています。

### FBAの仕組み

1. **在庫発送**: 出品者はまず、Amazonが提供する特定の梱包・配送ガイドラインに従って商品をフルフィルメントセンターに送ります。

2. **保管**: 商品がフルフィルメントセンターに到着すると、注文が入るまでAmazonが倉庫での保管を担当します。

3. **注文受付**: 顧客がFBA商品を注文すると、Amazonのシステムが自動的に取引を処理します。

4. **ピッキングとパッキング**: Amazonの従業員が棚から注文商品を取り出し、安全に梱包して発送の準備をします。

5. **配送**: 梱包された注文は、Prime配送を含むAmazonの迅速な配送オプションを利用して、直接顧客に発送されます。

6. **カスタマーサービス**: Amazonは返品や払い戻しの処理を含め、FBA注文に関連するカスタマーサービスの問い合わせも管理します。

### FBAの利点
1. **Primeバッジ**: 商品が自動的にPrimeの対象となる
2. **スケーラブルな運営**: 販売量の増加に容易に対応可能
3. **プロフェッショナルなフルフィルメント**: Amazonの物流ネットワークへのアクセス
4. **作業負荷の軽減**: 商品調達とマーケティングに集中可能

### FBAの欠点
1. **高いコスト**: 保管とフルフィルメントの手数料が利益率に影響を与える可能性がある
2. **コントロールの制限**: 在庫管理の監督が限定的
3. **長期保管料**: 動きの遅い在庫に対する追加料金
4. **複雑な準備規則**: 商品準備に関する厳格な要件

## FBA収益性の計算式

```

```

## FBM vs FBA 比較

| 機能 | フルフィルメント by マーチャント（FBM） | フルフィルメント by Amazon（FBA） |
|---------|--------------------------|------------------------|
| **在庫管理** | 保管、梱包、配送方法の完全なコントロール | Amazonが倉庫ネットワークを通じて保管と配送を管理 |
| **コスト構造** | 総手数料は低いが運営コスト（倉庫、スタッフ、配送）が高い | Amazon手数料は高いが保管、フルフィルメント、カスタマーサービスを含む |
| **配送速度** | 出品者の能力に応じて変動；通常3-7営業日 | Prime対象の1-2日配送；より速い配送時間 |
| **スケーラビリティ** | スケールするには重要なインフラと人員への投資が必要 | Amazonの広範なフルフィルメントネットワークで容易にスケール可能 |
| **Prime対象** | セラーフルフィルドPrimeを通じて可能だが、厳格な要件あり | すべての商品が自動的にPrime対象 |
| **カスタマーサービス** | 出品者がすべての顧客問い合わせ、問題、フォローアップを処理 | Amazonの24時間体制のカスタマーサービスチームがほとんどの問題を処理 |
| **保管コスト** | 倉庫/保管ソリューションに基づく固定費用 | 商品サイズ、重量、保管期間に基づく変動料金 |
| **返品処理** | 返品ポリシーと処理の完全なコントロール | Amazonがポリシーに従って返品を処理 |
| **在庫管理** | 在庫レベルと再注文のリアルタイムコントロール | 保管制限と料金により慎重な計画が必要 |
| **マルチチャネル販売** | 複数のプラットフォームでの在庫管理が容易 | マルチチャネルフルフィルメントには追加料金 |

## 適切な選択をする

FBMとFBAの選択時に考慮すべき要因：

1. **商品の特性**: 
   - **サイズと重量**: より重く大きい商品は保管・配送コストの増加によりFBA手数料が高くなる可能性があり、そのような商品ではFBMがより経済的な選択となる場合があります。対照的に、軽量で小型の商品はFBAの効率的な配送オプションの恩恵を受けやすいです。
   - **保管要件**: 生鮮品や壊れやすい商品など、特別な保管要件のある商品は、出品者が環境をコントロールできるFBMがより適している場合があります。FBAはすべての特殊な保管条件に対応できない可能性があります。
   - **利益率**: 商品の利益率を理解することは重要です。高利益率の商品はFBA手数料を吸収しやすい一方、低利益率の商品はFBAコストが高すぎると損失につながる可能性があります。

2. **ビジネスモデル**:
   - **販売量**: 大量販売の出品者は、Amazonの効率的な大量注文処理により規模の経済が働くため、FBAが有利な場合があります。対照的に、販売量の少ない出品者はFBMのコスト削減の恩恵を受けられる可能性があります。
   - **季節変動**: ビジネスが大きな季節変動を経験する場合、FBMは長期保管料を発生させることなく需要に応じて在庫レベルを調整できる、より柔軟な在庫管理を可能にします。
   - **成長計画**: 積極的な成長戦略を持つ出品者は、物流のための追加リソースを必要とせずに増加した注文量を処理できるスケーラビリティがあるため、FBAを好む場合があります。

3. **リソース**:
   - **利用可能な保管スペース**: スペースが限られている出品者は、自社の倉庫ソリューションを必要とせずにAmazonのフルフィルメントセンターに商品を保管できるFBAの恩恵を受けられます。これは大量の在庫を収容できない出品者に特に有益です。
   - **労働力の容量**: 梱包と配送を効率的に管理する人員が不足している出品者の場合、フルフィルメント業務をAmazonにアウトソーシングすることでこの負担を軽減し、ビジネスの他の側面に集中できるFBAが有益です。
   - **配送能力**: 確立された配送プロセスと運送業者との関係を持つ出品者はFBMを好む場合があります。これは物流の専門知識を活用できるためです。対照的に、堅固な配送能力を持たない出品者はFBAの包括的な物流サポートがより有益だと感じる可能性があります。

## よくある質問

**Q: FBMとFBAの両方を使用できますか？**
A: はい、多くの出品者が特に異なる商品タイプやピーク時期に、ハイブリッドアプローチを使用しています。

**Q: 初心者にはどちらの方法が適していますか？**
A: FBAは複雑な物流を処理するため初心者に適していることが多いですが、コストは慎重に検討する必要があります。

**Q: 各方式での返品はどのように処理されますか？**
A: FBM出品者は自社で返品を管理し、FBAの返品はAmazonのカスタマーサービスチームが処理します。

**Q: 手数料の主な違いは何ですか？**
A: FBMは手数料が低いですが、より多くの自己管理リソースが必要です。FBAは手数料が高いですが、包括的なフルフィルメントサービスが含まれています。

## 結論

FBMとFBAの選択は、Amazonビジネスのあらゆる側面に影響を与える重要な決定です。この選択を行う際は、商品タイプ、ビジネス目標、リソースを考慮してください。一つの方式で開始し、後で切り替えることも、両方を同時に使用して運営を最適化することも可能です。

重要なのは、特定の状況を徹底的に分析し、利用可能な計算ツールを使用してデータに基づいた決定を下すことです。FBMとFBAのどちらを選択しても、Amazonでの成功には慎重な計画、一貫した実行、そして変化する市場条件への適応性が必要です。
