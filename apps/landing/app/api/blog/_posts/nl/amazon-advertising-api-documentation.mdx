## Veelgestelde Vragen

A: Snelheidslimieten variëren per eindpunt en accounttype. Enterprise accounts hebben typisch hogere limieten, maar alle gebruikers moeten snelheidslimietafhandeling implementeren.

**V: Kan ik meerdere marketplace accounts beheren?**
A: Ja, de API ondersteunt multi-accountbeheer met juiste authenticatie- en machtigingsinstellingen.

**V: Hoe vaak kan ik rapporten genereren?**
A: Rapportgeneratiefrequentie is afhankelijk van het rapporttype en accountlimieten. De meeste rapporten kunnen per uur worden gegenereerd.

**V: Welke programmeertalen worden ondersteund?**
A: De API is taal-agnostisch en kan worden benaderd met elke taal die HTTP-verzoeken en JSON-verwerking ondersteunt.
