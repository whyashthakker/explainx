## COSMO와 RUFUS 이해하기

### COSMO: 백엔드 파워하우스
COSMO는 Amazon의 정교한 백엔드 AI 알고리즘으로:
- 대규모 언어 모델을 활용한 고객 검색 의도의 깊은 이해
- 방대한 고객 행동 데이터 분석을 통한 패턴과 선호도 파악
- 카탈로그 정보와 웹 데이터 처리로 상품 특성 종합적 파악
- 상식적 추론을 통한 상품 특징과 사용자 의도 연결

### RUFUS: 프론트엔드 어시스턴트
RUFUS는 대화형 쇼핑 어시스턴트로:
- 고객과 자연스러운 상호작용으로 원활한 쇼핑 여정 제공
- 개인 선호도와 이전 상호작용 기반 맞춤형 상품 추천
- COSMO의 인사이트를 활용한 추천 개선
- 인간다운 방식의 질문 응답

## 최적화 전략

### 컨텍스트 기반 최적화
1. **사용자 의도 분석**
   - 타겟 고객의 질문과 우려사항에 대한 철저한 조사
   - 상품이 해결할 수 있는 일반적인 문제점과 사용 시나리오 파악
   - 고객이 가장 중요시하는 상품 특징 파악

2. **콘텐츠 제작**
   - 특징뿐만 아니라 이점과 사용 사례를 포함한 포괄적인 상품 설명
   - 핵심 속성을 간단히 요약한 글머리 기호
   - 맥락에 맞고 시각적으로 매력적인 상품 이미지 최적화

### 백엔드 최적화
- 정확한 상품 속성 유지: 정기적 검토와 업데이트
- 플랫 파일 정기 업데이트: 재고 데이터 최신화
- 포괄적인 상품 데이터 제공: 사양, 치수, 사용 설명 등
- AI 기반 콘텐츠 생성 도구 활용
